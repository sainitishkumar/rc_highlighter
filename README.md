# rc_highlighter
A chrome extension for rosetta code to mark as TODO/Finished
Usage:

1. Download it as zip.
2. extract into a folder.
3. Open Chrome:
   a. Goto chrome://extensions/
   b. Toggle Developer mode
   c. Load Unpacked
   d. Open the folder you extracted it to.
